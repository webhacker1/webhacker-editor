# Документация проекта `editorWH`

Эта папка содержит практическую документацию по коду редактора:

1. `docs/01-quick-start.md`  
Как быстро запустить, собрать и протестировать проект.

2. `docs/02-architecture.md`  
Текущая архитектура: где `core`, где `features`, где `sanitize`, как проходит поток данных.

3. `docs/03-toolbar.md`  
Как устроен toolbar, где лежат кнопки, как добавить новую кнопку/дропдаун.

4. `docs/04-security.md`  
Модель безопасности: что санитизируется, где фильтруются URL/HTML, чего фронтенд не может гарантировать.

5. `docs/05-testing.md`  
Какие тесты есть, как запускать, как писать новые тесты.

6. `docs/06-recipes.md`  
Готовые рецепты: "добавить язык подсветки", "добавить пункт меню", "добавить обработчик события".

Рекомендуемый порядок чтения для новых разработчиков:

1. `docs/01-quick-start.md`
2. `docs/02-architecture.md`
3. `docs/03-toolbar.md`
4. `docs/04-security.md`
5. `docs/05-testing.md`
6. `docs/06-recipes.md`
