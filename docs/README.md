# Документация проекта

Эта папка — главная точка входа в документацию.

## Какие файлы читать
1. [01-quick-start.md](./01-quick-start.md) — быстрый запуск.
2. [02-architecture.md](./02-architecture.md) — архитектура и правила структуры.
3. [03-toolbar.md](./03-toolbar.md) — как добавлять кнопки и меню в toolbar.
4. [04-security.md](./04-security.md) — как работает sanitize и что нельзя ломать.
5. [05-testing.md](./05-testing.md) — тесты и чеклист проверки.
6. [06-integration-contract.md](./06-integration-contract.md) — публичный контракт интеграции.

## Документация по папкам
Во всех основных папках есть свой `README.md`: что это за слой, какой у него входной файл, что где лежит, и куда добавлять новый код.

## 5 минут до старта
1. Установить Node.js 20+.
2. Выполнить `npm install`.
3. Выполнить `npm run typecheck`.
4. Выполнить `npm test`.
5. Выполнить `npm run build`.
