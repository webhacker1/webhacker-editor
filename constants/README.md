# Константы

## Что это
Здесь лежат общие константы: клавиши, цвета, списки разрешенных тегов и языки кода.

## Точка входа
`indexConstants.ts`

## Что внутри
- `allowedTags.ts` — белый список тегов и служебные селекторы для sanitize.
- `codeLanguages.ts` — языки для подсветки кода.
- `colors.ts` — набор базовых цветов.
- `keyboard.ts` — общие коды/имена клавиш.

## Пример
```ts
import { KEY_ENTER } from "@/constants/indexConstants";
```

## Правила
- Только константы.
- Без бизнес-логики.
- Импортировать через `@/constants/indexConstants`.
