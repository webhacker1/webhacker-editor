# Ядро (Core)

## Что это
Базовый слой редактора: главный класс и ядро поведения.

## Точка входа
`indexCore.ts`

## Что внутри
- `WebHackerEditor.ts` — основной класс редактора.
- `commands.ts` — внешний запуск команд.
- `richtext/` — низкоуровневая логика редактирования.

## Пример
```ts
import WebHackerEditor from "@/core/indexCore";
```

## Правила
- Держим слой независимым от конкретного UI.
- Из других модулей заходим через `@/core/indexCore`.
