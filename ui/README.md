# UI-утилиты

## Что это
Небольшие переиспользуемые UI-хелперы.

## Точка входа
`indexUi.ts`

## Что внутри
- `elements.ts` — создание DOM-элементов.
- `theme.ts` — применение CSS-переменных темы.

## Пример
```ts
import { applyThemeVariables } from "@/ui/indexUi";
```

## Правила
- Только общий UI-слой без привязки к конкретной фиче.
- Не складываем сюда feature-логику.
