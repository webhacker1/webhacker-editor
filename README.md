## Что это

`WebHackerEditor` - библиотека rich-text редактора с:
1. toolbar;
2. code block + highlight.js;
3. table support;
4. sanitize для безопасного HTML.

## Установка

```bash
npm install
```

## Быстрый старт

```bash
npm run build
npm test
```
