# Типы

## Что это
Глобальные type declaration файлы проекта.

## Что внутри
- `modules.d.ts` — декларации для импортов модулей/ассетов.

## Пример
```ts
declare module "*.less";
```

## Правила
- Добавляем только то, что реально нужно сборке/TS.
- Избегаем слишком широких `any`, если можно указать точнее.
