# Стили

## Что это
Слой стилей редактора.

## Точка входа
`less/index.less`

## Что внутри
- `less/` — все Less-файлы по зонам UI.

## Пример
```ts
import "@/styles/less/index.less";
```

## Правила
- Глобальный импорт стилей только через `styles/less/index.less`.
- Компонентные/доменные стили раскладываем по отдельным less-файлам.
